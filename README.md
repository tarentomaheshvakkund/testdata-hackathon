# Test Data Repository for PR Review Hackathon

This repository contains test code files with various security vulnerabilities 
and code quality issues for testing the PR Review System.

## Purpose
- Generate test PRs with intentional issues
- Test automated code review agents
- Validate security and quality detection

## Categories
- Security Issues (SQL Injection, XSS, Command Injection, etc.)
- Code Quality Issues (Deep Nesting, Magic Numbers, Long Methods, etc.)

## Languages
- Python
- Java
